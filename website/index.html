Powered by Vector Tiles

Studio provides a new level of design control and interaction. All while being more performant, lighter, and faster - thanks to vector tiles. By packing layers, geometries, and feature attributes into lower zoom levels, vector tiles allow high zoom renders at a fraction of the storage cost of raster tiles. Paired with Mapnik and a CartoCSS stylesheet, vector tiles can be rendered as images, UTFGrids, geojson, and more.

Access to global data

Studio gives you instant streaming access to Mapbox's massive global datasets: Streets, Terrain, and Satellite. We manage all the data on our platform, providing live updates every minute to all our global data sets, and serve maps fast wherever you are in the world.

Style: Design

Style maps in Studio using CartoCSS stylesheets, basic metadata (name, description, attribution, etc.), and a reference to a source. Add custom fonts, patterns, and textures and export for print or publish to the web instantly.

Source: Data

Create custom vector tiles for your own datasets in Studio. Source projects contain configuration for converting a traditional geodata storage format (Shapefile, GeoJSON, etc.) into vector tiles. Once converted, the original datasource is no longer necessary, reference source with Mapbox access token and style layer by layer in Studio.
